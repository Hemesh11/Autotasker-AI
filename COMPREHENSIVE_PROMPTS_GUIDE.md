## ğŸ“‹ Prompt Patterns

### Pattern 1: Immediate Execution
**Format:** `[Action] [now/immediately/right now]`
```
Send me LeetCode questions now
Get my emails immediately
Summarize GitHub commits right now
```

### Pattern 2: Scheduled Time
**Format:** `[Action] [at/today at/tomorrow at] [TIME]`
```
Send me problems at 9am
Get emails today at 11:47pm
Summarize commits tomorrow at 2:30 PM
```

### Pattern 3: Daily Schedule
**Format:** `[Action] [daily/every day] [at TIME]`
```
Send me questions daily at 9am
Get emails every day at 8am
Generate problems daily at 10am
```

### Pattern 4: Interval Repetition
**Format:** `[Action] now [X] times with [Y] [unit] gap`
```
Send me questions now 3 times with 5 min gap
Get emails every 10 minutes, 5 times
Repeat 4 times with 15 minute interval
```

### Pattern 5: Combined Actions
**Format:** `[Action 1], [Action 2], and [Action 3]`
```
Get my emails, summarize them, and email the report
Fetch GitHub commits, analyze them, and send summary
Generate questions, email them, and log results
```
